Explicación del Modelo:

Esta predicción se genera utilizando un modelo de Regresión Logística. El modelo analiza patrones históricos de comportamiento del cliente, considerando variables como tipo de contrato, antigüedad, cargos mensuales, servicio de internet y soporte técnico.

Estadísticamente, los clientes con contratos mensuales, baja antigüedad y cargos elevados presentan mayor probabilidad de abandono del servicio.
